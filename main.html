VS Code cheat sheets

Basic editing

  • Ctrl+Shift+P or F1 - Show commanad line
  • !+Enter - Build empty html tamplate
  • Ctrl+Shift+Enter - Insert line above
  • Alt+Up/Down - Move line up or down
  • Shift+Alt+Up/Down - Copy line up or down
  • Crtl+Up/Down - Scroll line up or down
  • Alt+PgUp/PgDown - Scroll whole page up or down
  • Crtl+/ - Line comment
  • Shift+Alt+A - Block comment

Navigation

  • Ctrl+G - Go to line
  • Ctrl+P - Got to file
  • Alt+Left/Right - Go back/fowar
  • Ctrl+F - Find
  • Ctrl+H - Replace
  • Alt+Click - Insert cursor
  • Crtl+Alt+Up/Down - Insert cursor line above/below
  • Ctrl+Space - Sugestions
  • Ctrl+Shift+Space - Parameter hints
